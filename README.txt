There are four files in this submission.

The first is Feature Engineering.ipynb, which contains our feature engineering. 

Next is Linear Regression.ipynb which contains the linear regression performed on the dataset. 

Then there is Neural Network Implementation Testing, which is a file to demonstrate the neural network code is functional.

Finally there is Predicting House Prices with Our Neural Network.ipynb, which contains the performance of the neural network on the housing dataset.
